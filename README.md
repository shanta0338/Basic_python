# Basic Python

A collection of Python scripts created for learning and practising fundamental Python concepts. This repository covers core programming principles, object-oriented programming concepts, and built-in Python data structures.

## Overview

This project is designed as a practice repository for developing a strong foundation in Python programming. It includes examples of important Python concepts with simple implementations suitable for beginners.

## Features

- Python fundamentals
- Object-Oriented Programming (OOP)
- Data structures
- Functional programming concepts
- List comprehensions
- Lambda functions
- Practice exercises
- Beginner-friendly examples

## Installation

Clone this repository:

```bash
git clone https://github.com/shanta0338/Basic_python.git
```

Move into the project folder:

```bash
cd Basic_python
```

Run any Python file:

```bash
python filename.py
```

Example:

```bash
python inheritance.py
```
